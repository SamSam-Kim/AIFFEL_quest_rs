# 회고(참고 링크 및 코드 개선)

코드 리뷰를 하면서 불필요한 이미지 영역 처리를 줄이면 시간이 개선될 수 있겠다는 것을 느낄 수 있었습니다. 추가로 실제 이미지를 crop하면 이미지 로드 시간까지 조금 줄일 수 있지 않을까 생각했습니다.
문제 케이스와 결론 작성을 보고  코드만 신경쓰지말고 doc string이나 주석에 좀 더 신경을 써야겠다고 느꼈습니다.
